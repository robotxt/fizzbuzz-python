# FizzBuzz

## Requirements:

**Python:** 3.11.5

**Pytest** 

## Development setup:

**Setup Virtual Environment:**

`pyenv install 3.11.5`

`pyenv virtualenv 3.11.5 fizzbuzz`

`pyenv activate fizzbuzz`

**Install requirements:**

`pip install -r requirements.txt`

**Run FizzBuzz:**

`python main.py`


**Run Test:**

`./test.sh`

**Run Linter:**

`./linter.sh`
